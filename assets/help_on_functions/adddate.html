AddDate function

Actuarial Excel Addin

The AddDate function, returns a date that is t-Years away from Date. In order to do so, complete years are added first and the remaining fraction is converted into days by multiplying by 365.25.

See also zDur to calculate the years between two dates.

* : Required parameter

Parameters
Default
Description
Date *
--
.....
T-Years *
--
fraction and negative value are allowed

Example #1
t-Years is positive

Date = date(2016, 11, 3)

tYears = 4.6959

AddDate(Date, tYears) returns 2021-07-15


Example #2
t-Years is negative

Date = date(2021, 7, 15)

tYears = -4.6959

AddDate(Date, tYears) returns 2016-11-03
