MonthlyStat function

Actuarial Excel Addin

The MonthlyStat function returns a monthly statistics from the user's personal workbook my_aea.xlsm.

See also: YearlyStat, DailyStat

No required parameter

Parameters
Default
Description
Date
0
Date of the statistics, only year and month are considered, e.g. May 2, 2020 and May 17, 2020 will both return the May 2020 statistics
Stat No
0
Number assigned to the statistics, must be an integer, default value 0 returns the code's description
Code
0
See table below, default value 0 returns the statistics itself

Code
Description
0
Statistics
1
Name
2
Info type (integer, real number, boolean, text)
3
Description
4
Source
5
Start date
6
End date

Examples
Assuming StatNo=33, returns the number (in thousands) of passengers on Canadian airlines in a given month.

Date1 = date(2021, 9, 15)

MonthlyStat(Date1, 33) returns 2939


MonthlyStat(0, 33, 1) returns "Flight passengers CAN"

MonthlyStat(0, 33, 2) returns "integer"

MonthlyStat(0, 33, 3) returns "Number of passengers on Canadian airlines (x 1,000)"

MonthlyStat(0, 33, 4) returns "https://www150.statcan.gc.ca/t1/tbl1/fr/tv.action?pid=2310007901"

MonthlyStat(0, 33, 5) returns 199001

MonthlyStat(0, 33, 6) returns 202109


MonthlyStat(0, 0, 0) returns "Statistics"

MonthlyStat(0, 0, 1) returns "Name"

MonthlyStat(0, 0, 2) returns "Type"

MonthlyStat(0, 0, 3) returns "Description"

MonthlyStat(0, 0, 4) returns "Source"

MonthlyStat(0, 0, 5) returns "Start"

MonthlyStat(0, 0, 6) returns "End"
