YearlyStat function

Actuarial Excel Addin

The YearlyStat function returns a yearly statistics from the user's personal workbook my_aea.xlsm.

See also: MonthlyStat, DailyStat

No required parameter

Parameters
Default
Description
Year
0
Year of the statistics
Stat No
0
Number assigned to the statistics, must be an integer, default value 0 returns the code's description
Code
0
See table below, default value 0 returns the statistics itself

Code
Description
0
Statistics
1
Name
2
Info type (integer, real number, boolean, text)
3
Description
4
Source
5
Start date
6
End date

Examples
Assuming StatNo=102, returns the World population in a given year.

YearlyStat(2020, 102) returns 7794798739


YearlyStat(0, 102, 1) returns "World Population"

YearlyStat(0, 102, 2) returns "integer"

YearlyStat(0, 102, 3) returns "self explanatory"

YearlyStat(0, 102, 4) returns "https://www.worldometers.info/world-population/world-population-by-year/"

YearlyStat(0, 102, 5) returns 1951

YearlyStat(0, 102, 6) returns 2020


YearlyStat(0, 0, 0) returns "Statistics"

YearlyStat(0, 0, 1) returns "Name"

YearlyStat(0, 0, 2) returns "Type"

YearlyStat(0, 0, 3) returns "Description"

YearlyStat(0, 0, 4) returns "Source"

YearlyStat(0, 0, 5) returns "Start"

YearlyStat(0, 0, 6) returns "End"
