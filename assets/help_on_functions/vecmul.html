VecMul function

Actuarial Excel Addin

The VecMul function, multiplies each corresponding element of two vectors and returns a vector.

See also:

VecAdd, VecSub, VecDiv for other vector-vector transformation.

VecInv to invert the elements of a vector.

VecSum, VecProduct to transform a vector into a scalar.

ScalAdd, ScalSub, ScalMul, ScalDiv for vector-scalar transformation of a vector.

* : Required parameter

Example #1

.....

Vector1 = {9, 5, 7}

Vector2 = {2, 4, 6}

VecMul(Vector1, Vector2) returns {18, 20, 42}
