FmtNum function

Actuarial Excel Addin

The FmtNum function, takes a number as input and returns a text representing the number with or without the currency sign ($) and complying with the English or French number notation.

See also FmtDate, FmtMonth, FmtQuarter, MonthNum.

* : Required parameter

Parameters
Default
Description
Number *
--
Number beign formatted into a text
Decimal
0
Number of decimals kept, if negative, Number will be rounded by 10-Dec
Language
F
Either F for French or E for English
Currency sign
$ will be used if any character is provided

Example #1

Dec is positive

Number = 3141.59

Decimal = 1

Currency = "z"

FmtNum(Number, Decimal,, Currency) returns "3 141,6 $"

FmtNum(Number, Decimal, "E", Currency) returns "$3,141.6"


Example #2

Dec is negative

Number = 3141.59

Decimal = -2

FmtNum(Number, Decimal) returns "3 100"

FmtNum(Number, Decimal, "E") returns "3,100"
