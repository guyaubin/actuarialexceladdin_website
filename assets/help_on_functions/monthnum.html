MonthNum function

Actuarial Excel Addin

The MonthNum function, takes a text representation of a month as input and returns a number from 1 to 12 for the corresponding month.

See also FmtDate, FmtMonth, FmtQuarter.

* : Required parameter

Parameters
Default
Description
Month string *
--
e.g. Dec, December, dec, decembre, déc, décembre, ....

Example #1

.....

Month_string = "Jun"

MonthNum(Month_string) returns 6


Example #2

.....

Month_string = "jun."

MonthNum(Month_string) returns 6
