FindLastName function

Actuarial Excel Addin

The FindLastName function, returns the last name of a person.

See also FindFirstName, InvName.

* : Required parameter

Parameters
Default
Description
Name *
--
Full name with last name first then a separator then the first name
Separator
,
the character(s) separating the firstname and lastname

Example #1
Default separator (i.e. the comma)

Name = "Doe, John"

FindLastName(Name) returns "Doe"


Example #2
Special separator

Name = "Doe :: John"

Separator = "::"

FindLastName(Name, Separator) returns "Doe"
