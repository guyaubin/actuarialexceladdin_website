zDur function

Actuarial Excel Addin

The zDur function, returns the time elapsed (in years) between Date 1 and Date 2. Time elapse = complete years + remaining days divided by 365.

See also AddDate.

The Excel native function YearFrac also calculates the time elapsed between two dates.

* : Required parameter

Parameters
Default
Description
Date 1 *
--
.....
Date 2 *
--
.....
Negative allowed
False
False: Returns 0 if Date 2 is before Date 1

Example #1
Dates in chronological order

Date1 = date(2016, 11, 3)

Date2 = date(2021, 7, 15)

zDur(Date1, Date2) returns 4.6959


Example #2
Inverted dates

Date1 = date(2021, 7, 15)

Date2 = date(2016, 11, 3)

zDur(Date1, Date2) returns 0.0000


Example #3
Inverted dates, negative value allowed

Date1 = date(2021, 7, 15)

Date2 = date(2016, 11, 3)

NegativeAllowed = True

zDur(Date1, Date2, NegativeAllowed) returns -4.6959
