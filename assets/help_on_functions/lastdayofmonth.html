LastDayOfMonth function

Actuarial Excel Addin

The LastDayOfMonth function, returns a date, being the last day of a given month.

See also FirstDayOfMonth.

* : Required parameter

Parameters
Default
Description
Date *
--
.....
Working day
False
True: excluding Saturday and Sunday

Example #1
.....

Date = date(2021, 2, 15)

WorkingDay = False

LastDayOfMonth(Date, NextMonth, WorkingDay) returns 2021-02-28 (Sun)


Example #2
.....

Date = date(2021, 2, 15)

WorkingDay = True

LastDayOfMonth(Date, NextMonth, WorkingDay) returns 2021-02-26 (Fri)
