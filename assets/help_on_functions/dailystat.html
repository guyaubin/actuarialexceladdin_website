DailyStat function

Actuarial Excel Addin

The DailyStat function returns a daily statistics from the user's personal workbook my_aea.xlsm.

See also: YearlyStat, MonthlyStat

No required parameter

Parameters
Default
Description
Date
0
Date of the statistics
Stat No
0
Number assigned to the statistics, must be an integer, default value 0 returns the code's description
Code
0
See table below, default value 0 returns the statistics itself

Code
Description
0
Statistics
1
Name
2
Info type (integer, real number, boolean, text)
3
Description
4
Source
5
Start date
6
End date

Examples
Assuming StatNo=101, returns the exchange rate of the US$ / CA$.

Date1 = date(2021, 11, 29)

DailyStat(Date1, 101) returns 1.2765


DailyStat(0, 101, 1) returns "Exchange rate US$/CA$"

DailyStat(0, 101, 2) returns "real number"

DailyStat(0, 101, 3) returns "averages of aggregated price quotes from financial institutions published by 16:30 ET"

DailyStat(0, 101, 4) returns "https://www.bankofcanada.ca/rates/exchange/daily-exchange-rates/"

DailyStat(0, 101, 5) returns 2017-01-03

DailyStat(0, 101, 6) returns 2021-11-29


DailyStat(0, 0, 0) returns "Statistics"

DailyStat(0, 0, 1) returns "Name"

DailyStat(0, 0, 2) returns "Type"

DailyStat(0, 0, 3) returns "Description"

DailyStat(0, 0, 4) returns "Source"

DailyStat(0, 0, 5) returns "Start"

DailyStat(0, 0, 6) returns "End"
