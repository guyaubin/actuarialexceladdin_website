CleanDate function

Actuarial Excel Addin

The CleanDate function, takes either a string formatted date of 'real' date as input and returns the Excel date.

* : Required parameter

Parameters
Default
Description
Date *
--
Date (string or date format)
Format
--
A clue as the string formatted dates specifying the order for the Y (year), M (month) and D (day), e.g.: YMD, MDY, ...

Example #1
...

Date = date(1979, 7, 15)

Cleandate(Date) returns 1979-07-15
