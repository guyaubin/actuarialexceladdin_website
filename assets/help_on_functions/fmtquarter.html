FmtQuarter function

Actuarial Excel Addin

The FmtQuarter function, takes a date as input and returns a text representing the corresponding quarter either in French or English.

See also FmtDate, FmtMonth, FmtNum, MonthNum.

* : Required parameter

Parameters
Default
Description
Date *
--
....
Language
F
either F for French or E for English

Example #1

French

date = date(2021, 9, 20)

FmtQuarter(date) returns "3e trimestre"


Example #2

English

date = date(2021, 9, 20)

language = "E"

FmtQuarter(date) returns "3rd Quarter"
